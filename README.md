目录结构：

```
.
├── README.md	
├── dist	代码发布
├── docs	文档
├── gulpfile.js	Gulp 配置
├── src		源码
│   ├── html	模板
│   ├── js	JS
│   └── less	LESS
└── vendor	第三方库
```

Install：

```
git clone https://git.coding.net/ivymobi/Project-Fluke-Web.git

cd Project-Fluke-Web

npm install

http-server

浏览器里访问 http://127.0.0.1:8080
```
